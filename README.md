# FairChance

FairChance is a full-stack golf charity subscription platform where users can subscribe, select charities, submit scores, and participate in draw-based prize systems.

## Features
- User signup and login
- Role-based access for admin and subscriber
- Subscription management
- Charity selection
- Last 5 score tracking
- Draw generation
- Winner matching
- Admin dashboard

## Tech Stack
- React
- Tailwind CSS
- Node.js
- Express.js
- Supabase
